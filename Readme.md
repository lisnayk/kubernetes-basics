# Основи Kubernetes

Цей репозиторій містить навчальні матеріали курсу **«Основи Kubernetes»**, який присвячений вивченню принципів контейнеризації, 
архітектури Kubernetes та практичних підходів до розгортання, конфігурації й експлуатації застосунків у контейнеризованому середовищі.

Курс орієнтований на студентів спеціальностей **Комп’ютерні науки**, **Інженерія програмного забезпечення**, **Інформаційні системи** 
та всіх, хто хоче отримати ґрунтовне розуміння Kubernetes як платформи для сучасних хмарних застосунків.

---

## Мета курсу

Метою вивчення навчальної дисципліни «Основи Kubernetes» є формування у студентів системного розуміння принципів контейнеризації та оркестрації контейнерів, набуття практичних навичок розгортання, конфігурації та експлуатації застосунків у середовищі Kubernetes, а також ознайомлення з базовими підходами до масштабування, безпеки та моніторингу контейнеризованих систем.

**Після завершення вивчення дисципліни студент повинен знати:**

- принципи роботи контейнерів та контейнерної оркестрації;
- архітектуру Kubernetes та призначення його основних компонентів;
- призначення та структуру основних ресурсів Kubernetes;
- принципи мережевої взаємодії, конфігурації та зберігання даних у Kubernetes;
- базові підходи до безпеки та моніторингу контейнеризованих систем.

**Студент повинен вміти:**

- розгортати локальний Kubernetes-кластер;
- створювати та керувати Pod, Deployment, Service, ConfigMap, Secret;
- конфігурувати доступ до застосунків через Ingress;
- налаштовувати масштабування та health checks;
- аналізувати стан кластера та застосунків;
